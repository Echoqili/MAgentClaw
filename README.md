# MAgentClaw

多 Agent 管理系统，支持飞书、企业微信、Slack 等多种渠道接入。

## 安装

```bash
pip install maagentclaw
```

## 快速开始

```bash
python maagentclaw/main.py
```

访问 http://localhost:8000
