# MAgentClaw v1.2.0 发布说明

**发布日期**: 2026-03-08  
**版本**: v1.2.0  
**代号**: OpenClaw Integration

---

## 🎉 重大更新

v1.2.0 是一个里程碑式的版本，完整集成了 OpenClaw 的核心功能，并新增了 AI 模型支持、多渠道管理和增强的 Web 界面。

---

## ✨ 新增功能

### 1. AI 模型集成 🤖

**新增模块**:
- `maagentclaw/models/` - 完整的 AI 模型框架
- OpenAI GPT 系列支持
- 阿里云 Qwen 系列支持
- 模型管理器（支持自动回退）

**核心文件**:
- `base_model.py` - 模型基类
- `openai_model.py` - OpenAI 实现
- `qwen_model.py` - 阿里云实现
- `model_manager.py` - 模型管理

**功能特性**:
- ✅ 统一接口，易于切换
- ✅ 流式响应支持
- ✅ Token 计数和管理
- ✅ 自动回退机制
- ✅ 多模型并发

**使用示例**:
```python
from maagentclaw.models import create_openai_model

model = create_openai_model("gpt-3.5-turbo")
await model.initialize()

response = await model.chat([
    Message(role="user", content="你好")
])
```

---

### 2. 渠道管理 📡

**新增模块**:
- `maagentclaw/channels/` - 多渠道管理框架
- WebSocket 渠道（实时通信）
- REST API 渠道（HTTP 接口）
- CLI 渠道（命令行交互）
- 渠道路由器（智能路由）

**核心文件**:
- `base_channel.py` - 渠道基类
- `websocket_channel.py` - WebSocket 实现
- `rest_api_channel.py` - REST API 实现
- `cli_channel.py` - CLI 实现
- `channel_manager.py` - 渠道管理器
- `channel_router.py` - 渠道路由器

**功能特性**:
- ✅ 统一渠道接口
- ✅ 实时双向通信（WebSocket）
- ✅ RESTful API 支持
- ✅ 智能消息路由
- ✅ 多渠道并发

**使用示例**:
```python
from maagentclaw.channels import create_websocket_channel

channel = create_websocket_channel(port=8765)
await channel.initialize()

@channel.on_message
async def handle_message(msg):
    print(f"收到：{msg.content}")
```

**新增模块**:

**功能特性**:

- ✅ 工作空间文件编辑

**访问**: http://localhost:8000

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 设置环境变量

```bash
# OpenAI API Key
$env:OPENAI_API_KEY="your-key"

# 阿里云 DashScope Key
$env:DASHSCOPE_API_KEY="your-key"
```

### 3. 启动增强版

```bash
python maagentclaw/main_enhanced.py
```

### 4. 访问 Web 界面

http://localhost:8000

## 📖 使用指南

### AI 模型使用

```python
from maagentclaw.models import create_openai_model, Message

model = create_openai_model("gpt-3.5-turbo")
await model.initialize()

messages = [Message(role="user", content="你好")]
response = await model.chat(messages)

print(f"AI: {response.content}")
```

### 渠道使用

```python
from maagentclaw.channels import create_websocket_channel

channel = create_websocket_channel(port=8765)
await channel.initialize()

@channel.on_message
async def handle_message(msg):
    print(f"收到：{msg.content}")

await asyncio.sleep(3600)
```

3. 管理 Agent 和配置

5. 管理工作空间文件
6. 监控渠道状态
