## 二、新增模块详情

### 1. 浏览器自动化 (Browser Automation)

**来源**: OpenClaw、QClaw

**文件**: `maagentclaw/tools/browser_automation.py` (350 行)

**功能**:
- 浏览器启动和关闭
- 页面导航
- 元素点击
- 文本输入
- 下拉选择
- 悬停操作
- 页面滚动
- 等待元素
- 页面截图
- 元素获取
- JavaScript 执行
- 表单填写

**使用示例**:
```python
from maagentclaw.tools.browser_automation import BrowserAutomation, ElementSelector, ElementSelectorType

browser = BrowserAutomation(headless=True)
await browser.launch()

# 导航
await browser.navigate("https://example.com")

# 输入并点击
await browser.input_text(
    ElementSelector(ElementSelectorType.CSS, "#search"),
    "Python"
)
await browser.click(ElementSelector(ElementSelectorType.CSS, "#submit"))

# 截图
result = await browser.screenshot()
print(result.screenshot)  # base64

# 关闭
await browser.close()
```

### 2. 远程文件操作 (Remote File Operations)

**来源**: QClaw、WorkBuddy

**文件**: `maagentclaw/tools/remote_file_operator.py` (400 行)

**功能**:
- 目录列表
- 文件读取/写入
- 文件删除
- 文件复制/移动
- 目录创建
- 屏幕截图
- 命令执行
- 进程管理
- 剪贴板操作

**安全特性**:
- 路径白名单
- 文件大小限制
- 危险命令过滤

**使用示例**:
```python
from maagentclaw.tools.remote_file_operator import RemoteFileOperator

operator = RemoteFileOperator(Path("./workspace"))
operator.add_allowed_path(Path("/home/user"))

# 列出目录
result = await operator.list_directory("./data")
print(result.data["items"])

# 读取文件
result = await operator.read_file("./data/test.txt")

# 写入文件
await operator.write_file("./data/output.txt", "Hello World")

# 截图
result = await operator.get_screenshot()

# 运行命令
result = await operator.run_command("ls -la")
```

### 4. 增强定时任务 (Enhanced Scheduler)

**来源**: OpenClaw

**文件**: `maagentclaw/tasks/enhanced_scheduler.py` (400 行)

**功能**:
- Cron 表达式支持
- 固定间隔调度
- 单次执行
- 每日/每周/每月调度
- 任务重试
- 超时控制
- 执行历史
- 统计信息

**Cron 表达式**:
```python
# 每分钟
* * * * *

# 每天午夜
0 0 * * *

# 每周日午夜
0 0 * * 0

# 工作日早上9点
0 9 * * 1-5

# 每月1日午夜
0 0 1 * *
```

**使用示例**:
```python
from maagentclaw.tasks.enhanced_scheduler import EnhancedScheduler, ScheduleType, ScheduleTemplates

scheduler = EnhancedScheduler(Path("./workspace"))

# 添加 Cron 任务
scheduler.add_task(
    name="daily-report",
    command="generate-report",
    schedule_type=ScheduleType.CRON,
    cron_expression="0 0 * * *",
    description="每日报告生成"
)

# 添加间隔任务
scheduler.add_task(
    name="health-check",
    command="check-health",
    schedule_type=ScheduleType.INTERVAL,
    interval_seconds=60,
    description="健康检查"
)

# 使用模板
scheduler.add_task(
    name="workday-reminder",
    command="remind-meeting",
    **ScheduleTemplates.workdays_9am()
)

# 启动调度器
await scheduler.start()

# 获取统计
stats = scheduler.get_statistics()
print(stats["upcoming_tasks"])
```

## 五、使用示例

### 完整工作流

```python
from pathlib import Path
from maagentclaw.agents.multi_agent_orchestrator import MultiAgentOrchestrator, AgentMode
from maagentclaw.tasks.task_parser import TaskParser
from maagentclaw.tools.browser_automation import BrowserAutomation
from maagentclaw.tools.remote_file_operator import RemoteFileOperator
from maagentclaw.market.skills_marketplace import SkillsMarketplace
from maagentclaw.tasks.enhanced_scheduler import EnhancedScheduler

# 初始化
workspace = Path("./workspace")

# 1. 任务解析
parser = TaskParser(llm_callback)
intent = parser.parse("帮我搜索 AI 新闻并打开第一个结果")

# 2. 执行任务
orchestrator = MultiAgentOrchestrator(workspace, llm_callback)

# 3. 浏览器自动化
browser = BrowserAutomation()
await browser.launch()
await browser.navigate(intent.slots["url"])

# 4. 文件操作
operator = RemoteFileOperator(workspace)
await operator.write_file("./results.txt", browser.screenshot)

# 5. 定时任务
scheduler = EnhancedScheduler(workspace)
scheduler.add_task(
    name="daily-news",
    command="fetch-news",
    schedule_type="0 9 * * *"
)
await scheduler.start()

# 6. 安装技能
marketplace = SkillsMarketplace(workspace)
await marketplace.install_skill("github-assistant")
```
