## 核心功能详解

### 1. 心跳机制

**核心价值**: 实现 Agent 的周期性任务调度和状态监控

**关键特性**:
- HEARTBEAT.md 文件格式（人类可读）
- 心跳抑制逻辑（防止重复执行）
- 自动重试机制（失败任务）
- 完整的执行历史记录
- 实时统计信息

**使用示例**:
```python
# 创建 HEARTBEAT.md
# ## Task: health-check
# - Interval: 60
# - Command: check-health

# 启动心跳管理器
manager = HeartbeatManager(
    workspace_path,
    config,
    execution_callback=execute_function
)
await manager.start()
```

## 技术亮点

### 1. 异步编程

完全基于 asyncio 实现：

```python
# 心跳循环
async def _heartbeat_loop(self):
    while self.running:
        for task in self.tasks.values():
            if task.enabled and task.next_run <= datetime.now():
                await self._execute_task(task)
        await asyncio.sleep(self.config.interval)

# 技能执行
async def execute(self, **kwargs) -> SkillResult:
    # 异步执行技能逻辑
    result = await self.process(**kwargs)
    return SkillResult(success=True, data=result)
```

### 2. 数据类（Dataclasses）

使用 Python dataclasses 简化代码：

```python
@dataclass
class HeartbeatTask:
    name: str
    interval: int
    command: str
    enabled: bool = True
    last_run: Optional[datetime] = None
    next_run: Optional[datetime] = None
    status: TaskStatus = TaskStatus.PENDING

@dataclass
class SkillResult:
    success: bool
    data: Any = None
    error: Optional[str] = None
    duration: Optional[float] = None
    metadata: Dict[str, Any] = field(default_factory=dict)
```

### 3. 抽象基类（ABC）

定义清晰的接口：

```python
class BaseSkill(ABC):
    @abstractmethod
    async def execute(self, **kwargs) -> SkillResult:
        pass

class BaseChannel(ABC):
    @abstractmethod
    async def send(self, message: ChannelMessage) -> bool:
        pass
    
    @abstractmethod
    async def receive(self) -> Optional[ChannelMessage]:
        pass
```

### 4. 动态模块导入

灵活的技能加载：

```python
def load_from_file(self, file_path: Path) -> Optional[BaseSkill]:
    spec = importlib.util.spec_from_file_location(
        "skill_module", file_path
    )
    module = importlib.util.module_from_spec(spec)
    spec.loader.exec_module(module)
    
    # 查找技能类
    for attr_name in dir(module):
        attr = getattr(module, attr_name)
        if (isinstance(attr, type) and 
            issubclass(attr, BaseSkill) and 
            attr != BaseSkill):
            return attr()
```

### 5. 配置管理

统一的配置模式：

```python
@dataclass
class HeartbeatConfig:
    enabled: bool = True
    interval: int = 60
    max_retries: int = 3
    retry_delay: int = 5
    suppress_duplicates: bool = True

@dataclass
class SkillConfig:
    enabled: bool = True
    timeout: int = 30
    max_retries: int = 0
    rate_limit: Optional[int] = None
```
